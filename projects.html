NASA Hunch Cybersecurity
The goal of this project was to design a Reliable, Redundant, and Secure Network to transport the data, video, telemetry, and voice from the moon's Smart Building to a Control Center(s) on Earth. For this we created a trifold presentation, 3 network designs demonstrating the possible architecture to send data to the moon, a demonstration on how possible laser communications could work to deliver this data, along with a website with our comprehensive write ups to showcase this work.

FEDA Education Website

The organization works to improve financial literacy among College and High School level students. It is currently partnered with the University of Texas at Dallas. With them the website is constantly being updated with new financial tips and advice to help those in need.

TFT Task Manager
The intended usage of this product was to help members of the non-profit organization Toys For Texans better manage the delegation of tasks and ensure deadlines are met. It is built in REACT and uses Flask API. It takes in picture authentication to allow for managers and VPs to ensure tasks are completed on time and correctly. This overall allowed for a more streamlined and efficent running of the organization.
